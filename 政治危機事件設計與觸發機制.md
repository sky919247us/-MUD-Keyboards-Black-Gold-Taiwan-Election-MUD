# 政治危機事件設計與觸發機制

> **模組定位**：本文件為 `00-game-architecture.md` 第九章的完整展開，詳述所有突發事件的觸發條件、數值影響與玩家應對選項。

## 一、基礎 5 大危機事件（原版）

### 危機 1：基礎設施跳電

| 欄位 | 內容 |
|------|------|
| **連動 API** | 天氣 API（高溫炎熱）＋ PTT API（停電/跳電聲量激增） |
| **觸發條件** | 氣象 API 顯示「高溫」且 PTT 偵測到「停電/跳電」關鍵字聲量破爆 |
| **事件描述** | 極端高溫導致虛擬市區大跳電。在野黨對手在漆黑街道上開直播，抨擊執政黨能源政策破產。 |
| **核心影響** | 執政黨好感度大扣；必須消耗「幕僚行動力」啟動網軍，帶風向將停電定調為「電網老舊」而非「缺電」。 |
| **應對選項** | A. 消耗 AP 發布聲明（吸收 50% 傷害）；B. 消耗政治獻金購買主流媒體版面（吸收 80% 傷害）；C. 放任（全額承擔） |

### 危機 2：豆腐渣工程現形

| 欄位 | 內容 |
|------|------|
| **連動 API** | 天氣 API（大雨/豪雨特報）＋ 水利署水位 API（水位超警戒） |
| **觸發條件** | 現實氣象 API 發布大雨或豪雨特報，且監測站水位異常 |
| **事件描述** | 連日暴雨導致市區新建國民運動中心天花板漏水、擋土牆滑落。對手陣營召開記者會，指市長收受「建商」黑金回扣。 |
| **核心影響** | 仇恨值暴增；若 JSON 陣列中擁有建商勢力，需進行「切割判定」，否則支持度每小時持續失血。 |
| **切割判定** | 若玩家主動宣布與建商切割：獻金池立即扣減 20%，但仇恨值增加減半 |

### 危機 3：經濟恐慌與無薪假風暴

| 欄位 | 內容 |
|------|------|
| **連動 API** | 股市大盤 API（台股單日跌幅 > 3%） |
| **觸發條件** | 台股單日跌幅超過設定閾值 |
| **事件描述** | 大盤重挫引發企業恐慌，市區知名科技大廠宣布無薪假。在野黨發起「討生活、顧顧佛祖」大遊行。 |
| **核心影響** | 全體選民「經濟投票」懲罰啟動，執政黨流失中間選民；未來 3個 Tick 內政治獻金收入減半。 |
| **應對選項** | 執政黨可消耗 60 AP 啟動「經濟振興方案」緊急聲明，抵銷 40% 獻金衰減 |

### 危機 4：1450 側翼反串翻車

| 欄位 | 內容 |
|------|------|
| **連動 API** | PTT API（爆卦文推文數超過爆 XX 等級）＋ Google Trends（急升） |
| **觸發條件** | 玩家使用網軍陣列發動帶風向任務，RNG 判定「被鄉民抓包」（隱蔽度不足時機率大幅提升） |
| **事件描述** | 網軍側翼帳號在 PTT 發布帶風向文，因忘記切換 IP 被鄉民抓包，史稱「大型翻車現場」。 |
| **核心影響** | 網軍陣列整體隱蔽度強制歸零；好感度轉化為對手悲情牌加成；必須犧牲一位高階幕僚辭職止血。 |
| **傷害公式** | `aggro_gain = output_power × 2`；`favorability_loss = stealth_remaining × 15` |

### 危機 5：颱風假決策的政治豪賭

| 欄位 | 內容 |
|------|------|
| **連動 API** | 天氣 API（颱風警報，但風雨預測處邊緣值） |
| **觸發條件** | 中央氣象署發布陸上颱風警報，但預測強度處於「宣布/不宣布颱風假」的邊緣閾值 |
| **事件描述** | 鄰近縣市都宣布颱風假，唯獨玩家轄區因科學數據未達標宣布「正常上班上課」，結果當天狂風暴雨，民眾臉書灌爆憤怒表情。 |
| **核心影響** | 「災難政治學」最高懲罰等級；好感度瞬間暴跌 2000-4000 點。 |
| **隱藏選項** | 消耗鉅額政治獻金（≥ 500萬）購買媒體版面，強行洗版「市長尊重專業，不向民粹妥協」，可抵銷 60% 傷害，但增加法務風險值 +15 |

## 二、新增危機事件（API 擴充版）

### 危機 6：消防車陷阱（連動：消防署災害應變中心 API）

| 欄位 | 內容 |
|------|------|
| **連動 API** | 消防署應變中心 API（二級以上開設）＋ 天氣 API |
| **觸發條件** | 應變中心開設且撤離勸離人數超過特定閾值，且玩家為地方首長層級 |
| **事件描述** | 豪雨引發山區土石流，多處民宅遭到掩埋。應變中心緊急啟動，前線消防隊卻因玩家未事先「預防性撤離佈署」而陷入混亂。民眾在社群媒體上瘋傳「消防車陷在泥堆裡」的畫面。 |
| **核心影響** | `favorability -= 2500`；`aggro += 2000`；幕僚 AP 恢復速率降低 50%（持續 24 小時） |
| **應對選項** | A. 消耗 40 AP 親赴災區直播（吸收 30% 傷害，但增加知名度 +500）；B. 消耗大量政治獻金調度額外人力（吸收 60% 傷害） |

---

### 危機 7：掃黑連累——黑金陣列反噬（連動：警政 110 / 治安 API）

| 欄位 | 內容 |
|------|------|
| **連動 API** | 警政署掃黑專線 API（特定轄區掃黑通報飆高）＋ PTT API |
| **觸發條件** | 玩家陣列中含有黑道/樁腳類勢力（如「海線砂石互助會」），且現實世界同一地區發生大規模掃黑行動 |
| **事件描述** | 警方突擊掃黑，直播中赫然出現與玩家有關的樁腳頭目被押送畫面。對手陣營截圖狂發：「候選人背後的黑金真面目！」 |
| **核心影響** | 被連結的樁腳節點：`loyalty → 0`，`mobilization_power` 暫時凍結 48 小時；玩家：`favorability -= 1800`，`legal_risk_index += 25` |
| **應對選項** | A. 立即宣布與該樁腳切割（減少 60% 傷害，但永久失去該樁腳）；B. 花費 300 萬政治獻金聘請律師「發表無關聲明」（僅吸收 20% 傷害，保留樁腳但法務風險仍累積） |

---

### 危機 8：缺電政治核彈（連動：台電 API）

| 欄位 | 內容 |
|------|------|
| **連動 API** | 台電即時電力 API（備轉容量率 < 6%）＋ PTT API（停電聲量） |
| **觸發條件** | 台電 API 顯示「供電吃緊」或「限電警戒」，且發生實際停電 |
| **事件描述** | 大停電正值晚間用電尖峰，家庭主婦、上班族與電商商家集體在 PTT 與 IG 開砲。在野黨宣布召開全國緊急記者會，點名執政黨各縣市候選人——「你們的能源政策把台灣帶進黑暗！」 |
| **核心影響** | 執政黨所有層級玩家全域 `favorability -= 1000 ~ 2500`（依停電規模）；在野黨/永續綠黨網軍攻擊力 × 200%（持續 4 Tick） |
| **隱藏觸發** | 若同時有「高溫」天氣 API 觸發，額外疊加「酷暑停電雙重懲罰」：所有陸軍掃街行動力消耗 × 2 |

---

### 危機 9：積淹水官商勾結風暴（連動：水利署 API）

| 欄位 | 內容 |
|------|------|
| **連動 API** | 水利署水位 API（監測站水位超警戒）＋ 天氣 API（大雨/極端降雨） |
| **觸發條件** | 監測站水位在 30 分鐘內暴漲超過設定閾值，且玩家陣列中含有建商勢力 |
| **事件描述** | 強降雨導致對手市區大淹水，媒體找到圖資顯示淹水最嚴重路段，正是玩家管轄的「新開發重劃區」——由盟友建商承包。調查記者追出，該路段的防汛工程竟被轉包三次，工程費被抽走 40%。 |
| **核心影響** | `aggro += 3000`；`favorability -= 2000`；結盟建商陣列所有節點觸發「切割判定」RNG |
| **應對選項** | A. 主動切除建商陣列所有節點（傷害降低 70%，但獻金池損失 30%）；B. 啟動側翼網軍洗版「洪水是天災，非人禍」（高風險，隱蔽度消耗 × 1.5） |

### 危機 10：空污選票崩盤（連動：環境部 AQI API）

| 欄位 | 內容 |
|------|------|
| **連動 API** | 環境部 AQI API（紅色以上）＋ PTT/Google Trends（空污議題聲量） |
| **觸發條件** | 特定區域（尤以中南部）AQI 連續 3 小時以上處於紅色警示（151+），且網路空污相關聲量破爆 |
| **事件描述** | 中南部紫爆，天空呈橘紅色。由永續綠黨側翼主導的環保公民串聯發起「拒絕毒霾，拒絕爛政策」快閃行動。傳統陸軍的掃街活動被迫縮水，因為沒有選民願意在爛空氣中聽政見。 |
| **核心影響** | 傳統政黨執政滿意度每小時 `favorability -= 200`（持續至 AQI 恢復）；陸軍掃街行動力消耗 × 2；永續綠黨玩家獲得「空污紅利」：知名度自然增長 × 3 |
| **應對選項** | A. 消耗 AP 召開環保記者會（止血每小時好感扣減）；B. 花費政治獻金強推線上廣告，將資源轉移至空戰 |

## 三、跨 API 複合危機事件

### 複合危機 A：太平洋三連擊（天氣 × 台電 × 水利）

**觸發條件**：同一 Tick 內，颱風 API 警報 ＋ 台電備轉容量率低迷 ＋ 水位警戒同時觸發

**效果**：
- 所有地方首長玩家進入「超高壓緊急模式」
- 無論任何指令，AP 消耗 × 3
- 若執政黨玩家未能在 2 小時內（現實時間）發布「緊急整合聲明」，將觸發「全面治理崩潰」事件，好感度與仇恨值同時往極端移動 3000 點

---

### 複合危機 B：政商媒鐵三角崩盤（股市 × 新聞 API × 法務風險值）

**觸發條件**：台股重挫 ＋ 負面新聞情感分數 < -0.7 ＋ 玩家 `legal_risk_index > 60`

**效果**：
- 系統自動觸發「政治獻金法查核事件」
- 玩家所有政治獻金池立即凍結 48 小時
- 敵方可在此期間發動「一擊必殺」的法務攻擊，強制扣除玩家 50% 現有獻金並沒收
