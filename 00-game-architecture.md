## 一、設計哲學與核心定位

本遊戲是一款以**台灣現代選戰**為背景的硬核文字 MUD（Multi-User Dungeon）政治模擬遊戲。核心創新點在於「**現實驅動狀態演化系統（Reality-Driven State Evolution System）**」：

- 虛擬政治實體透過 **Webhook / Cron Job** 與現實世界 API 強制耦合
- 不可抗的外部事件（颱風、股災、輿論爆料）即時改寫遊戲內數值
- 玩家在瞬息萬變的現實環境中做資源分配、損害控制與戰略抉擇

## 四、核心屬性與邊界懲罰機制

| 屬性 | 區間 | 歸零 / 觸底懲罰 |
|------|------|----------------|
| **知名度 (fame)** | 0–10,000 | 「政治透明人」：零自然流量、小額募款鎖死 |
| **好感度 (favorability)** | -10,000–10,000 | 0 → 信賴危機；-10,000 → 樁腳流失 5/hr、罷免成功率 95% |
| **仇恨值 (aggro)** | 0–10,000 | 0 → 話題絕緣體，網軍攻擊無效；10,000 → 全國下架運動 |
| **政治獻金** | 0–2B NTD | 0 → 糧草斷絕：幕僚 AP 鎖死、廣告全下架 |
| **幕僚行動力 (AP)** | 0–100 | 0 → 過勞：自動防禦關閉；強行行動有 75% 觸發公關災難 |

### 特殊隱藏狀態
- **造神運動**：好感度極高時觸發，道德標準閾值提升，微小負面事件 → 指數崩跌
- **法務風險值**：暗盤收受政治獻金時累積，超過閾值 → 金額沒收 + 刑事懲罰

## 五、陣列資產系統（非對稱作戰）

### 5.1 地方樁腳 (local_bosses)
```json
{
  "boss_id": "boss_001",
  "region_code": "TPE-DA",
  "mobilization_power": 1500,
  "loyalty": 75
}
```
- `loyalty < 30`：敵方可發動「**拔樁**」，成功後動員力轉移至敵方

### 5.2 網軍側翼帳號 (cyber_army_accounts)
```json
{
  "node_id": "army_ptt_01",
  "platform": "PTT",
  "stealth_rating": 80,
  "output_power": 500
}
```
- 超高頻攻擊消耗 `stealth_rating`；歸零 → 「**網軍被抓包**」公關危機，引發對手悲情牌加成

### 5.3 自媒體陣列 (media_channels)
```json
{
  "channel_id": "yt_001",
  "platform": "YouTube",
  "subscribers": 200000,
  "alignment_type": "Allied"
}
```
- **Owned**：無損傳遞政見
- **Allied**：擴展同溫層（負面事件時有切割機率）
- **Paid**：持續消耗政治獻金，停止付款即停播

## 七、現實 API 聯動邏輯

### 7.1 股市 API（TWSE）
監控大盤指數日變動率 `Δ%`：

- `Δ% < -3%`（股災）→ 執政黨好感度扣 `500 × |Δ%|`；全域政治獻金縮減 40%；在野黨網軍攻擊力 × 150%
- `Δ% > +3%`（大漲）→ 執政黨好感度 + 加成；企業獻金意願提升

### 7.2 天氣 API（中央氣象署 W-C0034-001）
解析 `areaDesc`（影響縣市）+ 關鍵字比對（豪雨/颱風/土石流）：

- 地方首長若未在警報前 12hr 消耗 AP 執行「防颱視察」→ 災情應對指數降至 0.5
- 好感度結算：`Δfav = baseDmg × 應對指數 × 責任係數`
- 自動觸發中央↔地方「**卸責攻防事件**」（雙向網軍結算）

### 7.3 新聞情感 API
加權情緒分數 `W_sentiment`（-1.0 ~ +1.0）× 聲量熱度：

- `W > 0.5 且聲量極高`（正面國際事件）→ 抗中保台標籤候選人好感大增
- `W < -0.5 且聲量爆表`（醜聞/貪腐）→ 連鎖崩潰：
  - 仇恨值飆升
  - 好感度崩盤
  - Allied 媒體頻道 RNG 切割判定
  - 所有樁腳忠誠度強制 -20

### 7.4 內政部消防署 / 地方災害應變中心 API

串接數據：災害應變中心二級（含）以上開設時，提供**避難收容所開設統計、撤離勸離人數、受損情形**等資料。

**遊戲應用（災防動員與首長大考驗）**：
- 應變中心開設 → 地方首長玩家強制進入「緊急狀態」
- 若未及時消耗 AP 發布安撫與調度指令 → 觸發「救災不力」公關危機
- 好感度與中間選民支持度大幅滑落

### 7.5 警政署與地方警察局 110 報案 / 治安 API

串接數據：110 報案系統、掃黑專線、路霸交通違規檢舉、重大刑案通報。

**遊戲應用（治安風暴與黑金反噬）**：
- 特定轄區重大刑案或掃黑通報頻率異常飆高 → 在野黨玩家可發動「治安敗壞」輿論攻擊
- 若玩家陣列中有「海線砂石互助會」等黑金樁腳，一旦現實爆發掃黑專案 → 該樁腳「動員力」暫時凍結，並連帶扣除玩家好感度

### 7.6 台電即時電力與停電資訊 API

串接數據：每日尖峰備轉容量率、即時各機組發電量、計畫性或突發性停電區域。

**遊戲應用（能源政策攻防與跳電危機）**：
- 備轉容量率低迷 → 系統廣播「缺電警戒」
- 實際停電 + PTT 怨氣指數飆高 → 在野黨與永續綠黨網軍獲得巨額攻擊力加成
- 執政黨必須花費大量政治獻金進行危機公關

### 7.7 經濟部水利署即時水位 API

串接數據：全台中央管河川之每 10 分鐘更新一次即時水位觀測資料。

**遊戲應用（治水工程與豆腐渣現形）**：
- 與天氣 API 搭配使用
- 短延時強降雨導致監測站水位暴漲 → 觸發「地方積淹水」事件
- 若玩家長期收受建商政治獻金，對手可召開記者會指控「官商勾結、防汛偷工減料」 → 高仇恨值懲罰

### 7.8 環境部空氣品質指標（AQI）API

串接數據：即時 AQI 觀測資料。

**遊戲應用（空污紅害與環保選票）**：
- 中南部 AQI 達紅色警示 → 觸發「空污紫爆」狀態
- 傳統政黨執政滿意度按小時扣血
- 陸軍掃街拜票行動力消耗加倍（空污導致民眾不滿且團隊易過勞）
- 迫使玩家將資源轉移至網路空戰

## 八、AI 結算引擎（System Prompt 設計）

### 輸入格式
| 變數 | 說明 |
|------|------|
| `Player_State` | 玩家當前 JSON 狀態（屬性 + 資源 + 陣列） |
| `Reality_API` | 現實 API 快照（股市、天氣、新聞） |
| `Player_Action` | 玩家指令字串 |

### 輸出格式（固定 JSON）
```json
{
  "news_report": "100-150 字聳動台灣新聞台旁白（帶媒體立場偏誤）",
  "ptt_comments": [
    "推: 鄉民正向或反諷留言",
    "噓: 鄉民攻擊留言",
    "→: 中立吃瓜留言"
  ],
  "state_changes": {
    "favorability": 0,
    "aggro": 0,
    "fame": 0,
    "political_funds": 0,
    "staff_ap": 0,
    "array_events": "陣列影響描述"
  }
}
```

### 防幻覺機制
- 禁止直接引用真實政治人物名字
- 使用「某黨高層、執政黨發言人、王姓建商、李姓樁腳」等虛擬代稱
- 現實 API 數據視為**客觀事實不可扭曲**，但可根據玩家陣營進行「政治解讀」

## 十、每日核心循環（Game Loop）

```
每日循環
│
├─ 1. 登入收成（Passive Income）
│     └─ 樁腳離線小額獻金 + 自媒體流量分潤 + 知名度自然增長
│
├─ 2. 消耗行動力（Energy Sink）
│     ├─ 低 AP：跑紅白帖/掃街 → +樁腳忠誠度，+微量好感度
│     ├─ 中 AP：錄影片/發文開砲 → +知名度（+仇恨值如含攻擊）
│     └─ 高 AP：上政論節目連線 → 高風險高報酬，結算依當日 API 風向
│
├─ 3. 現實 API 廣播（每 10 分鐘 Tick）
│     └─ 股市 / 天氣 / 新聞 狀態更新，可能觸發突發事件
│
└─ 4. 資源枯竭 / 冷卻
      └─ AP 耗盡 → 進度停滯，產生心理付費暗示
```
